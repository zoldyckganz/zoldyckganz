Deskripsi Proyek

Aplikasi ini adalah web "Dompet Kos" yang merupakan sistem pencatatan keuangan sederhana. Sistem ini dapat digunakan untuk mencatat, menampilkan, mengubah, dan menghapus (CRUD) data pemasukan dan pengeluaran. Aplikasi ini juga dilengkapi fitur pencarian transaksi serta kalkulasi otomatis untuk menghitung total pemasukan, total pengeluaran, dan sisa saldo akhir.

Alasan Memilih Tema

Saya memilih tema ini karena sangat relevan dengan kebutuhan anak kos atau mahasiswa untuk mengelola keuangan. Selain itu, tema ini sangat ideal untuk menerapkan semua materi backend yang dipelajari, terutama fungsionalitas CRUD (Create, Read, Update, Delete) dengan database MySQL, serta mengimplementasikan logika PHP untuk fitur pencarian dan kalkulasi saldo.
